A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na contrução de páginas na Web. Documentos HTML podem ser interpretado por navegadores.

Alguns dos recursos da lingugaem HTML vistos até este momento:


Recurso Tags Associadas Descrição
Links <a> Criar links entre paginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar lista de itens nao ordenados
Imagens <img> Inserir imagens
